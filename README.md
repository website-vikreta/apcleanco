# ApcleanCo

A minimal Next.js 14 static website for ApcleanCo.

## Getting Started

### Development
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build
```bash
npm run build
```

The static site will be generated in the `out` directory.

## Project Structure

- `app/` - Next.js App Router pages
- `public/` - Static assets
- `next.config.js` - Configuration with static export enabled
